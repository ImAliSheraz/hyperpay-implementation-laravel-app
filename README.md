# Laravel Hyperpay boilerplate

To Start testing the [laravel-hyperpay](https://github.com/devinweb/laravel-hyperpay)

1- Migrate the database.

```shell
php artisan migrate
```

2- Update the `.env`

```shell
# HyperPay env keys

SANDBOX_MODE=true
ENTITY_ID=
ENTITY_ID_MADA=
ACCESS_TOKEN=
CURRENCY=SAR
```

3- Create your first user.

4- Navigate to the route `/hyperpay/checkout`.
